## Benefits

### Strengthen security

Strengthen security by storing user data on AWS instead of on vulnerable endpoint devices.  

### Cost effective

Fixed-rate pricing, with no overprovisioning, and no upfront costs. You only pay for what you use.  

### Scale on demand

Available at scale when you need it, streamlining IT operations.  

### Maximize productivity

Maximize remote worker productivity with reliable, high-performance, globally distributed infrastructure.
